# 3D Renderer

This repository contains learning resources and code examples for building a 3D renderer from scratch.  
It is designed to help developers understand the fundamental concepts of 3D graphics pipelines and rendering techniques, 
by providing CPU-based implementation in rust, and using only a single pixel drawing primitive.
